Description of the Workflow

After DNA sequencing, automated programs and scripts submit the sequence reads for demultiplexing, followed by analysis in the targetted exome pipeline. The pipeline is implemented using two programs; the sns program which implments a standard exome variant calling pipeline tailored for implementation at NYU’s phoenix compute cluster, and the snsxt program which included extra analysis and reporting steps customized for usage in the NGS580 gene panel.
